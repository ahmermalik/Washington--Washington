[Washington, Washington](http://gamejolt.com/games/platformer/washington-washington/39341/)
======================

Game written in HTML5/JavaScript using pixi.js.

Controls
======================
- Arrow keys to move
- Spacebar to jump
- Z to melee
- X to shoot
- 1 - Pistol
- 2 - SMG
- 3 - Shotgun
